# Mapdeck

Mapdeck is a universal geospatial visualization and analysis platform that connects data layers with analytical tools through standardized APIs.

## Built-in Widgets

* **MapViewer** - Interactive map visualization
* **AttributeTable** - Data table for spatial attributes
* **LayerTree** - Hierarchical layer management with search

## Quick Start

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`

> **Note:** Example data (`public/`) is not included in the repository. To run the app with sample data generate your own STAC catalog.
